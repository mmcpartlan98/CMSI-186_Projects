Scope notes: public variables, classes, and methods can be accessed by from outside the class. Private means that it can only be accessed from within the calling class or method.

Try-Catch Statement

Try block: a collection of tests to run.
Catch block: collecting errors
Finally block: Execute at the end of the test program to handle any exceptions that may have been thrown.
Other error checking: the assert statement.

assert condition : exception_to_throw;
